# Document RAG v0.2.0

本版本将产品名称更新为 **Document RAG**，Windows 主程序更名为 `DocQA.exe`。

## 主要更新

- 新增 GitHub Release 自动检查、跳过版本、稍后提醒和手动检查更新。
- 新增独立 `Updater.exe`，支持下载进度、SHA-256 双重校验、数据库备份和失败回滚。
- 更新程序文件时永久保留 `user-data`、知识库、Qdrant 索引和本地模型。
- 支持随版本更新 `DocQA.exe` 和 `docqa.ico`，并刷新 Windows 图标缓存。
- 问答模型更新为本地 Qwen3，保留 BGE 中文嵌入和混合检索能力。
- 完善文档、表格、演示文稿和压缩包解析、检索去重、上下文扩展与答案证据审查。
- 优化网页界面、文档状态、字段定位、原文验证和知识库管理。

## 下载

下载 `DocQA-v0.2.0-win-x64.zip`，完整解压后运行 `DocQA.exe`。

首次启动或首次使用模型时可能需要联网下载本地模型。应用仅监听本机
`127.0.0.1`，用户数据默认保存在程序目录的 `user-data` 中。

## 从 v0.1.0 升级

已发布的 v0.1.0 尚未包含自动更新器，因此本次需要手动下载并解压 v0.2.0。
可以将旧安装目录的 `user-data` 整体移动到新目录，知识库、索引和模型会被保留。
从 v0.2.0 之后可以直接在应用内检查和安装更新。
