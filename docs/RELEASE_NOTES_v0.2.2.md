# Document RAG v0.2.2

## 升级验证版本

此版本用于验证从 `v0.2.1` 到 `v0.2.2` 的完整自动更新流程：

- 启动后可通过“设置 -> 检查更新”发现 `v0.2.2`；
- 标准在线版自动下载标准更新包；
- 离线完整版自动下载包含模型与解析工具的离线更新包；
- 下载时同时校验 GitHub asset digest 和 `.sha256`；
- 安装前备份 SQLite，更新失败时自动回滚；
- 更新程序、图标和版本信息，永久保留整个 `user-data`；
- 保留知识库、授权目录、SQLite、Qdrant 向量索引和标准版模型缓存；
- 离线版继续提供 BGE、Reranker、Qwen3、llama.cpp 和 libarchive。

本版本没有更换向量模型、切块规则、数据库结构或索引格式，因此不需要重新建立知识库
或重新生成向量。

## 下载

- `DocQA-v0.2.2-win-x64.zip`：标准在线版；
- `DocQA-v0.2.2-win-x64-offline.zip`：离线完整版。

两个包均为完整首次安装包，也可由 `v0.2.1` 的内置更新器直接安装。
